## Project Overview

This project focuses on identifying phishing emails and differentiating them from legitimate communications by analyzing sender information, email content, hyperlinks, urgency tactics, and social engineering indicators.

The assessment was conducted using phishing awareness resources and practical email analysis techniques to improve cybersecurity awareness and promote safe email practices.

---

## Objectives

- Identify phishing indicators.
- Analyze legitimate and phishing emails.
- Assess security risks.
- Improve cybersecurity awareness.
- Promote safe email practices.

---

## Email Samples Analyzed

### Legitimate Emails

- Dropbox Storage Notification
- UPS Delivery Notification

### Phishing Emails

- Fake Google Docs Share
- Coca-Cola Prize Scam
- Fake Google Password Alert

---

## Key Findings

- Suspicious sender domains
- Fake brand impersonation
- Urgency and fear tactics
- Credential theft attempts
- Social engineering techniques
